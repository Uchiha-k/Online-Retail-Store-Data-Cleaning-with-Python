 # Online-Retail-Store-Data-Cleaning-with-Python
Automating the cleaning of messy Excel data (blanks, unwanted rows, date formatting, revenue calculation).
Steps Covered:
- Remove blanks
- Flag/remove unwanted rows
- Standardize column names
- Add Revenue column
- Format Invoice Date
- Export cleaned dataset
- Tools Used: Python, Pandas, Jupyter Notebook
Next Steps: Feed cleaned data into Tableau/Power BI for visualization
